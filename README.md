# لعبة تخمين الأرقام العربية - الإصدار 2D

لعبة تخمين أرقام تفاعلية مع واجهة عربية قابلة للاستجابة. اللاعبون يحاولون تخمين رقم سري يتلقون تعليقات حول دقة التخمين. اللعبة تدعم الوضع الفردي والجماعي مع التواصل الفوري عبر WebSocket.

## الميزات الرئيسية

- **وضع فردي**: العب ضد الحاسوب مع مستويات صعوبة مختلفة
- **وضع جماعي**: لعب تنافسي فوري مع لاعبين آخرين عبر الإنترنت
- **تحديات متعددة**: 4 تحديات صغيرة للحصول على تلميحات:
  - تحدي الذاكرة: تذكر تسلسل الألوان
  - تحدي الاتجاهات: ترتيب الأسهم في الاتجاهات الصحيحة
  - تحدي حبات المطر: حل معادلات رياضية سريعة
  - تحدي التخمين: توقع النمط الصحيح
- **نظام البطاقات**: بطاقات فريدة حصرية للعب الجماعي مع تأثيرات خاصة
- **واجهة عربية كاملة**: تصميم يدعم اللغة العربية (يمين لليسار)
- **تصميم متجاوب**: واجهات منفصلة للهاتف والحاسوب مع تحسين لكل جهاز
- **نظام إعادة الاتصال**: الحفاظ على حالة اللعبة عند قطع الاتصال لمدة 5 دقائق

## البنية المعمارية

### Frontend (الواجهة الأمامية)

**التقنيات المستخدمة:**
- React 18 مع TypeScript للمكونات
- Vite كأداة البناء والتطوير
- TailwindCSS للتصميم المتقدم
- Radix UI للمكونات الوصول إليها
- Zustand لإدارة حالة التطبيق

**التخطيط:**
- **ديسكتوب**: واجهة كاملة مع لوحة المحاولات على الجانب
- **موبايل**: واجهة محسنة للشاشات الصغيرة مع لوحة أرقام مخصصة
- **كشف الجهاز**: اكتشاف تلقائي عند 768px

**مكونات الحالة:**
- `useNumberGame`: إدارة حالة اللعبة الرئيسية
- `useChallenges`: إدارة حالة التحديات الصغيرة
- `useCards`: نظام البطاقات للعب الجماعي
- `useAudio`: تأثيرات صوتية وملاحظات سمعية

### Backend (الخادم)

**التقنيات المستخدمة:**
- Express.js مع TypeScript
- WebSocket (ws) للتواصل الفوري
- Neon PostgreSQL مع Drizzle ORM
- جلسات محلية الذاكرة

**منطق اللعبة:**
- نظام الغرف مع معرفات فريدة
- تبادل الأكواز السرية بين اللاعبين
- نظام التوقيت (60 ثانية لكل دور)
- نظام التحقق من الإجابات
- إدارة تلقائية لانقطاعات الاتصال

## نموذج البيانات

- **جدول المستخدمين**: للمستقبل (authentication جاهز لكن غير مفعل)
- **إدارة الجلسات**: تتبع WebSocket وحالة الغرف
- **الحفاظ على الحالة**: حفظ محلي في sessionStorage للاتصال بعد التحديث

## التبعيات الخارجية الرئيسية

**واجهة المستخدم:**
- `@radix-ui/*`: مكونات UI آمنة وسهلة الوصول
- `react-icons`: أيقونات عالية الجودة
- `framer-motion`: حركات سلسة

**إدارة الحالة:**
- `zustand`: إدارة حالة خفيفة الوزن
- `@tanstack/react-query`: إدارة حالة الخادم

**التصميم:**
- `tailwindcss`: نظام تصميم حديث
- `tailwind-merge` و `clsx`: أدوات فئات شرطية

**الأصوات:**
- `howler`: مشغل صوتي عالي الأداء

**التطوير:**
- `tsx`: تنفيذ TypeScript مباشر
- `vite`: بناء سريع مع HMR
